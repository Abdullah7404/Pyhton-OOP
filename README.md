# Python OOP Logic Building Practice

This repository contains beginner friendly Python programs focused on **Object Oriented Programming (OOP)** and **logic building**.  
The goal of this project is to practice Python concepts using simple, real life examples.

---

## Project Overview

This repository includes small Python classes that help in understanding:
• How classes and objects work  
• How to use constructors (`__init__`)  
• How to store and access data using instance variables  
• How to apply conditions and logic inside methods  

Each example is kept simple so beginners can easily follow the flow.

---

## Concepts Used

• Classes and Objects  
• Constructor (`__init__`)  
• Instance Variables  
• Methods  
• Conditional Statements  
• Basic Data Handling
